# Instructions to Modify CommNet Constellation

1. GameData/CommNetConstellation
	- replace the original cnc_settings.cfg with the one provided here
	
2. If you are installing this over over top of a previous saved game you need to do the following; 
	- edit the persistent.sfs file and delete the entire SENARIO name = CNCCommNetScenario and restart your game 
	
	
Notes: If you upgrade CommNet Constellation you will need to repeat this process to get the correct settings for this mod
