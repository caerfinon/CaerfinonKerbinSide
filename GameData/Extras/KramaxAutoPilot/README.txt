# Instructions to Modify Kramax Auto Pilot

1. GameData/KramaxAutoPilot
	- replace the original DefaultFlightPlans.cfg with the one provided here
	
2. GameData/KramaxAutoPilot/Extras
	- delete the flightplans.cfg as it contains old original Kerbinside locations
	
Notes: If you upgrade Kramax Auto Pilot you will need to repeat this process to get the correct flightplan settings for this mod
