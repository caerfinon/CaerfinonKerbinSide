# Instructions to Modify NavInstruments

1. GameData/NavInstruments/Plugindata
	- replace the original defaultRunways.cfg with the one provided here
	- delete the makingHistoryRunways.cfg they are merged with the one provided here
	- You can keep or delete the Helipads.cfg depending on you preference
	
2. GameData/NavInstruments/ModuleManagerCfgs
	- delete the KerbinSideRemastered.cfg as it contains Kerbin Side Remastered locations
	- delete the JNSQ.cfg if you want. It only takes effect if JNSQ is installed
	- No not delete the glideslopes.cfg
	
Notes: If you upgrade NavInstruments you will need to repeat this process to get the correct flightplan settings for this mod
